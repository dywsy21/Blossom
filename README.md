# To build:
1. clone this repository

```bash
git clone https://github.com/dywsy21/Graph-Max-Matching.git
```

2. cd to this directory
3. Run the following commands:

**For Linux:**
```bash
mkdir build && cd build
cmake ..
make -j$(nproc)
```

**For Windows:**
```bash
mkdir build && cd build
cmake ..
cmake --build . --config Release
```
